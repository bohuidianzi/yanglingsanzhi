| 问题 | 具体表现 | 影响页面 |

## 9. 色彩体系优化方案

### 9.1 V1 → V2 色值对照

| 角色 | V1色值 | V2色值 | 变更理由 |
|------|--------|--------|----------|
| 主色-深蓝 | `#1A3A5C` | `#0F2B47` | 更深邃沉稳，提升高级感，减少蓝色明度跳跃 |
| 主色-浅蓝 | `#2E6B9E` | `#1E4D73` | 与主色渐变更自然，减少与白的对比 |
| 主色-light | `#3D5B8B` | `#2A5A8A` | 中间色阶微调，过渡更柔和 |
| 辅助-科技绿 | `#2E7D32` | `#3A8F5C` | 降饱和度，更沉稳科技感，减少与深蓝的冲突 |
| 点缀-橙 | `#E8912B` | `#D4862A` | 微降饱和度，避免突兀，保留CTA辨识度 |
| 背景-浅灰 | `#F5F7FA` | `#F8F9FC` | 微调更冷灰，配合深蓝主色 |
| 新增-文字深灰 | - | `#2D3748` | 正文主色，替代纯黑`#000` |
| 新增-文字中灰 | - | `#718096` | 次要文字，替代`#666` |
| 新增-文字浅灰 | - | `#A0AEC0` | 辅助说明文字 |
| 新增-分割线 | - | `#E2E8F0` | 统一分割线/边框色 |

### 9.2 Tailwind 色阶更新

```
primary: { 500: '#0F2B47', 400: '#1E4D73', 300: '#2A5A8A', ... }
success: { 500: '#3A8F5C', 400: '#4FA874', ... }
secondary: { 500: '#D4862A', 400: '#E89B3E', ... }
neutral: { 900: '#2D3748', 600: '#718096', 400: '#A0AEC0', 200: '#E2E8F0', 50: '#F8F9FC' }
```

## 10. 交互动效规范

| 交互元素 | 动效类型 | 参数 | 优先级 |
|----------|----------|------|--------|
| 页面滚动-元素渐入 | translateY(40px)→0 + opacity 0→1 | duration: 0.6s, ease: easeOut | P0（已有，保留） |
