# 杨凌三智官网 V2 - 产品系列卡片升级

## 已完成

### 问题
五大产品系列卡片颜色太花——蓝、青、橙、紫、绿五张高饱和渐变色块放一起像彩虹糖，与下方三方资源协同的极简风格也不协调。

### 方案：莫兰迪灰调 + 动态JS效果

**配色升级**（全部降饱和，统一灰调）：
| 产品 | 旧色 | 新色 |
|------|------|------|
| 水蚀监测 | `#1565C0→#1E88E5` 亮蓝 | `#4A6FA5→#6B8DB8` 蓝灰 |
| 风蚀监测 | `#00838F→#00ACC1` 亮青 | `#5A8A8A→#7AA5A5` 青灰 |
| 气象监测 | `#E65100→#FF8F00` 亮橙 | `#B8834A→#C99E6B` 暖棕 |
| 盐碱治理 | `#7B1FA2→#AB47BC` 亮紫 | `#7B6B8D→#968AA6` 紫灰 |
| 智慧平台 | `#2E7D32→#43A047` 亮绿 | `#5A7D5A→#7A9B7A` 绿灰 |

**动态效果**（纯 CSS @keyframes，无额外依赖）：
1. **浮动大圆呼吸** — 封面内两张半透明大圆球缓慢上下浮动+缩放，每张卡片独立延迟，形成波浪韵律
2. **同心圆环旋转** — 右上角双圆环 20s 反向缓慢旋转，左下角单圆环 15s 正向旋转
3. **hover 光带扫过** — 鼠标悬停时一道半透明白光从左到右扫过卡片（shimmer）
4. **渐变角度差异化** — 五张卡片各用不同渐变角度（145°/155°/135°/165°/150°），打破单调

**细节收敛**：
- 毛玻璃图标 80px→64px，更克制
- 装饰元素透明度整体降低 40%
- 底部色点 8px→6px，操作区更紧凑
- 标题 letter-spacing 改为 -0.2，更现代
- 圆点阵背景 opacity 0.08→0.06
